# Java Interactive Chatbot

A modern and interactive chatbot written in Java with a beautiful graphical user interface.

## Features

- Modern and attractive GUI using Java Swing
- Real-time chat interface with message history
- Timestamp for each message
- Natural language processing for basic conversations
- Pattern matching for responses
- Greeting and farewell detection
- Customizable responses

## How to Compile and Run

1. Make sure you have Java installed on your system
2. Compile the Java files:
   ```
   javac ChatBot.java ChatBotGUI.java
   ```
3. Run the program:
   ```
   java ChatBotGUI
   ```

## Usage

- Type your messages in the input field at the bottom of the window
- Press Enter or click the Send button to send your message
- The chat history will be displayed in the main window
- Each message includes a timestamp
- The bot will respond to various inputs including:
  - Greetings (hello, hi, hey, etc.)
  - Questions about its capabilities
  - Basic conversation
  - Jokes
- Type 'quit' or 'exit' to end the conversation

## Example Conversations

You can try these example inputs:
- "Hello"
- "What is your name?"
- "How are you?"
- "Tell me a joke"
- "What can you do?"
- "Who created you?"
